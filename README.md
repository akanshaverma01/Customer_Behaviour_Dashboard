# 📊 Customer Behavior Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer behavior to extract meaningful insights that can help businesses improve decision-making, increase customer retention, and optimize sales strategies.

The analysis was performed using **Python, SQL, and Power BI**, covering the complete data analysis workflow from data cleaning to visualization.

---

## 🎯 Objectives

* Understand customer purchasing behavior
* Identify high-value customers
* Analyze sales trends and patterns
* Detect customer retention and churn behavior

---

## 🛠️ Tools & Technologies Used

* **Python** (Pandas, NumPy) – Data cleaning & preprocessing
* **SQL** – Data querying and transformation
* **Power BI** – Interactive dashboard creation

---

## 🔄 Project Workflow

1. **Data Collection**

   * Imported raw customer dataset

2. **Data Cleaning (Python)**

   * Removed missing and duplicate values
   * Standardized data formats
   * Prepared dataset for analysis

3. **Data Analysis (SQL)**

   * Wrote queries to extract insights
   * Aggregated customer and sales data
   * Created structured views for reporting

4. **Data Visualization (Power BI)**

   * Built interactive dashboard
   * Used charts, KPIs, and slicers for insights

---

## 📊 Key Insights
 * Clothing category generates the highest sales and revenue, making it the top-performing segment.
 * Majority of customers (73%) are non-subscribers, indicating a big opportunity to improve subscription conversions.
 * Average purchase amount (~$59.8) is moderate, suggesting scope for upselling or bundling strategies.
 * Young and middle-aged customers contribute the most revenue, making them the key target audience.
 * Average review rating (3.8) is decent but not excellent, highlighting room for improving customer satisfaction.
